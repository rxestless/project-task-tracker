# project-task-tracker
Idea taken from -> https://roadmap.sh/projects/task-tracker

A lightweight Command-Line Interface (CLI) task manager built with Python. It allows users to create, view, search, update, delete, and mark tasks as completed, with automatic backend tracking for creation and modification timestamps.

# ✨ Features
- **Add Tasks:** Create new tasks with a title, description, and due date.
- **List Tasks:** Display all existing tasks with their current completion status.
- **Search Task Details:** View detailed information for a specific task.
- **Searching Tasks based on their Status :** View information of all the tasks based on their status (Complete, Incomplete, In-Progress)
- **Update Tasks:** Edit titles, descriptions, and due dates on the fly.
- **Mark Complete:** Instantly flip task status to complete.
- **Delete Tasks:** Remove unwanted tasks by their index.
- **Automatic Timestamps:** Backend tracking for `createdAt` and `updatedAt` on every modification.
- **Persistent Storage:** Saves all task data cleanly into a local `tasks.json` file.
